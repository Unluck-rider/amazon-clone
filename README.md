# Amazon Clone (UI Only and fake api integration)

This is a simple Amazon clone built using Flutter. It includes the following pages with only UI design (no backend or API integration):

- Login Page
- Forgot Password Page
- Register Page
- Home Page
- Profile Page
- Cart Page
- Menu Page
